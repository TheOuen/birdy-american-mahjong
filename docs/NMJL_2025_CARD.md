# NMJL 2025 Card — All Valid Winning Hands

Source: Mahjong Sequence Final.docx

**Legend:**
- `F` = Flower
- `D` = Dragon
- `N/E/W/S` = Wind (North/East/West/South)
- Numbers = suit tiles (Bam, Crak, or Dot)
- `0` = matching zero (same number as adjacent group in opposite suit)
- `X` = Exposable hand
- `C` = Concealed hand required
- Point value follows X/C designation
